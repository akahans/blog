Build static website based on pelican

tags: blog python pelican github

What is pelican

Pelican is a static site generator that requires no database or server-side logic.

Environment

  • MacOs
  • Python 3.6
  • PyCharm 20xx

Install pelican

1.Create venv on your computer You can create in command line:

python3.6 -m venv ~/PycharmProjects/Personal/pelican/venv

Creating Virtual Environment Configure a virtual environment PyCharm

2.Enter venv

source ~/PycharmProjects/Personal/pelican/venv/bin/activate

3.Install pip package

pip install pelican markdown ghp-import

Initialization your blog

1.Run command

pelican-quickstart

then into interactive mode, fill what pelican want to know, you can modify them in pelicanconf.py

2.After all fill done, run command

make html && make serve

the dev server will run on port 8000, click http://127.0.0.1:8000 you can preview your site.

Write post

Write this post at ~/pelican/content/ with markdown format.

Release new article

Run command

make html

the static file will generate under ~/pelican/output/

Publish new post

Hosted directly from GitHub repository will create a new repo, then upload ~/pelican/output/* to this public repo.
